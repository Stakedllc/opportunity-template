# Opportunity Template
## Installation
```bash
# Git Clone
git clone git@github.com:Stakedllc/opportunity-template.git

# Change Directory
cd opportunity-template

# Install node project dependencies
npm i
```

#### Using hardhat
  - npx hardhat accounts
  - npx hardhat compile
  - npx hardhat test
  - npx hardhat node
  - node scripts/sample-script.js
  - npx hardhat help
